# ⚽ Worldcup 2026 Experience

Une plateforme web moderne et interactive dédiée aux fans de football, conçue pour offrir une expérience immersive autour de la Coupe du Monde 2026.

---

## 🚀 À propos du projet

Worldcup 2026 Experience est une application frontend développée avec des technologies modernes, mettant l’accent sur :

* une **expérience utilisateur immersive**
* un **design UI/UX soigné**
* une **architecture évolutive prête pour un backend sécurisé**

---

## 🎯 Objectifs

* Offrir une expérience fluide et engageante
* Mettre en valeur les moments clés du football
* Simuler des fonctionnalités interactives (prédictions, statistiques)
* Concevoir une base prête pour une application complète

---

## 🎨 Design & Interface

Interface conçue sur Figma avec une approche moderne :

* Hero section immersive
* Layout responsive basé sur des cards
* Animations fluides et interactions dynamiques
* Design system cohérent

👉 Design original :
https://www.figma.com/design/oGvyZXFFzEzWCmeNO5RMKX/Landing-Page-for-World-Cup

## ⚙️ Installation & Démarrage

### 1. Cloner le projet

```
git clone https://github.com/amiineed/Worldcup2026Experience.git
cd Worldcup2026Experience
```

### 2. Installer les dépendances

```
npm install
```

### 3. Lancer le projet

```
npm run dev
```

## 🔐 Sécurité

Ce projet étant actuellement **frontend-only**, les mécanismes de sécurité complets ne sont pas encore implémentés.

Cependant, l’architecture est pensée pour intégrer :

* Authentification sécurisée (JWT / sessions)
* Validation des données côté serveur
* Protection contre les injections (XSS, SQL)
* API sécurisée
