# ⚽ Worldcup 2026 Experience

Une plateforme web moderne et interactive dédiée aux fans de football, conçue pour vivre l'expérience de la Coupe du Monde 2026 comme jamais auparavant.

---

## 🚀 À propos du projet

**Worldcup 2026 Experience** est une application web de nouvelle génération qui combine design UI/UX avancé et une architecture technologique robuste pour créer une expérience immersive autour de la Coupe du Monde 2026.

### Objectifs principaux

- 🎯 Offrir une expérience utilisateur intuitive et engageante
- 🏟️ Mettre en avant les moments clés du tournoi
- 🌍 Créer une communauté mondiale de fans
- 📊 Fournir des fonctionnalités interactives et des prédictions
- 🔐 Assurer la sécurité des données utilisateur

---

## 🎨 Design & Interface

L'interface a été entièrement conçue sur **Figma** avec une approche mobile-first et moderne :

- **Hero section immersive** avec visuels captivants
- **Layout responsive** basé sur des cartes et grilles
- **Typographie hiérarchisée** pour une meilleure lisibilité
- **Composants interactifs** avec animations fluides
- **Design System cohérent** utilisant Radix UI et Material UI

👉 **Design original** : [Figma - World Cup Landing Page](https://www.figma.com/design/oGvyZXFFzEzWCmeNO5RMKX/Landing-Page-for-World-Cup)

## 🚀 Installation & Démarrage

### Prérequis
- Node.js ≥ 16.x
- npm ou pnpm

### 1. Cloner le repository
```bash
git clone https://github.com/amiineed/Worldcup2026Experience.git
cd Worldcup2026Experience
2. Installer les dépendances
bash
npm install
# ou avec pnpm
pnpm install
3. Lancer le serveur de développement
bash
npm run dev
L'application sera accessible sur http://localhost:5173 (port Vite par défaut)

4. Builder pour la production
bash
npm run build
